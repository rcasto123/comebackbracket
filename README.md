# comebackbracket

A lightweight NCAA March Madness bracket comeback tracker infographic built with vanilla JavaScript.

## What it does

comebackbracket lets you visualize comeback moments across an NCAA March Madness bracket — highlighting which teams overcame deficits or long odds to advance through each round. It presents bracket data as an infographic, making it easy to see at a glance which matchups had the most dramatic reversals.

## Tech stack

- Vanilla JavaScript
- HTML5 / CSS3
- No build step or external dependencies required

## How to use

1. Clone or download the repository.
2. Open `index.html` in your browser.
3. The infographic will render the bracket data directly in the page.

No server required — everything runs client-side.
